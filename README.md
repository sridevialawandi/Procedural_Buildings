<h3>Procedural Generation of Buildings</h3>

The building footprint shapes are: O,S,T,F,L and C.

Press “t” to apply Perlin Noise to the rooftop plates.

Extra components added are
* Chimney
* Air Went
* Street Light
* Swings
* Lawns
* Flower pot on some window sills
